# Welcome Recruit

![Copilot Studio Agent Academy
Recruit](../images/mcs-agent-academy-recruit-banner.png)

**리크루트에 오신 것을 환영합니다.**\
이 미션을 받아들이신다면, 당신의 목표는 **Microsoft Copilot Studio**를
활용하여 에이전트를 구축하는 기술을 마스터하는 것입니다.

이 실습 중심 교육은 **에이전트의 세계**로 들어가는 시작점입니다.
grounded 프롬프트부터 Adaptive Cards, 그리고 에이전트 플로우까지, 실제
도구와 사용 사례를 통해 지능형 에이전트를 구축하고 확장하며 배포하는
방법을 배우게 됩니다.

## 🎯 미션 목표

Agent Academy를 완료하면 다음을 할 수 있습니다:

-   Microsoft Copilot Studio 맥락에서 에이전트가 무엇인지 이해하기
-   대형 언어 모델(LLM), 검색 증강 생성(RAG), 그리고 오케스트레이션이
    에이전트 안에서 어떻게 결합되는지 탐색하기
-   **선언형(declarative)** 에이전트와 **커스텀(custom)** 에이전트 모두
    구축하기
-   **Topics**, **Adaptive Cards**, **Agent Flows**를 활용해 에이전트
    기능 확장하기
-   에이전트를 **Microsoft Teams** 및 **Microsoft 365 Copilot**에
    배포하기

## 🧪 사전 준비 사항

모든 미션을 완료하려면 다음이 필요합니다:

-   Microsoft 365 Developer 테넌트 (SharePoint 활성화 필요)
-   **Microsoft Copilot Studio** 접근 권한 (체험판 또는 라이선스)
-   선택 사항: SharePoint, Power Platform, Power Fx에 대한 기본 지식

## 🧬 대상 사용자

이 과정은 다음 사용자에게 적합합니다:

-   **Copilot Studio**를 탐색하는 메이커 및 개발자
-   **Microsoft 365 Copilot 확장**을 구축하는 IT 전문가
-   지능형 에이전트로 **한 단계 더 성장하고 싶은** Power Platform 사용자
-   **직접 실습하며** 배우는 것을 선호하는 모든 사람
